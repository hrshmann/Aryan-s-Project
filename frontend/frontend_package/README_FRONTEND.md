# Mental Health App — Frontend

## Tech Stack
- React 19 + TypeScript
- Vite (build tool)
- Tailwind CSS v4
- Recharts (data visualization)
- Lucide React (icons)
- Motion (animations)

## Components
- `AuthPage.tsx` — Login / Register
- `LandingPage.tsx` — Home/landing screen
- `Header.tsx` / `Footer.tsx` — Layout
- `Chatbot.tsx` — AI-powered chat interface
- `MoodTracker.tsx` — Mood logging & history
- `WellnessTrackers.tsx` — Wellness metrics
- `SelfCare.tsx` — Self-care tools with audio
- `Forum.tsx` — Community forum
- `TherapistBooking.tsx` — Book therapist sessions
- `AdminTherapistDashboard.tsx` — Admin/therapist panel

## Setup
```bash
npm install
npm run dev       # Start dev server
npm run build     # Build for production
```

## Environment
Copy `.env.example` to `.env` and set:
- `APP_URL` — Your frontend URL
